| :---------------: | :----------: | :-------------: |
|      0(128)       |      0       |       &30       |
|      1(129)       |      1       |       &31       |
|      2(130)       |      2       |       &32       |
|      3(131)       |      3       |       &33       |
|      4(132)       |      4       |       &34       |
|      5(133)       |      5       |       &35       |
|      6(134)       |      6       |       &36       |
|      7(135)       |      7       |       &37       |
|      8(136)       |      8       |       &38       |
|      9(137)       |      9       |       &39       |
|      10(138)      |      .       |       &2E       |
